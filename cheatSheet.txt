View all databases :
show bds;

create ea new or switch databases:
use dbName

view current Database:
db

Delete Database :
db.dropDatabase();

Create a collection named 'myComments' 
db.createCollection('myComments')

Create a collection named 'myComments' 
db.myComments.drop()

*** MongoDb commands for rows:-

Show all Rows in a collection==
db.comments.find()

Find the first row matching the object==
db.comments.findOne({name: 'rhb'})


insert on Row ==
db.comments.insert({
    'name': 'rhb',
    'lang': 'JS',
    'member_since': 5
})

inset many rows== 
db.comments.insertMany([{
    'name': 'rhb',
    'lang': 'JS',
    'member_since':5
},
{
    'name': 'rakib',
    'lang': 'Python',
    'member_since': 4
},
{
    'name': 'rohan',
    'lang': 'ruby',
    'member_since':5
}
])

Show all Rows in a collection (Prettified)===
db.comments.find().pretty()



Search in a MongoDB database ==
db.comments.find({'lang': 'python'})

Limit the number of rows in output==
db.comments.find().limit(2)

Count the number of rows in output==
db.comments.find().count()

Sorting Database==
db.comments.find().sort({}).pretty()


Update  a row==

db.comments.update({'name': 'rhb'},
{
    'name': 'rhb',
    'lang': 'JS',
    'member_since': 51
}, {upsert: true})

MongoDB Increment Operator ==

db.comments.update({'name': 'rhb'},
{$inc:{
    member_since: 2
}})


MongoDB Rename Operator ==

db.comments.update({'name': 'rhb'},
{$rename:{
    member_since: member
}})


Delete Row ==
db.comments.remove({'name': 'rhb'})


Less than==
db.comments.find({member_since: {$lt: 90}})


Greater than==
db.comments.find({member_since: {$gt: 90}})
